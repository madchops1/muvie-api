VISUALZ is

Passionate about technology, visual arts, music & artists. The Visualz home base is Chicago.

VISUALZ founders combined these passions into a platform of tools for artists to create realtime, audio reactive visuals using web technologies such as WebGL, Web Audio, WebRTC, and custom shaders for live events, concert visuals, streams, interactive installations, & experiences.

We build software, real-time audio-reactive music visualization tools, live concert visuals, and interactive installations.
