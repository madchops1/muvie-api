Learn How It Works

Crash Course:

Welcome to VISUALZ. When you open the app it will create a track for you automatically. The left menu toggles between the Track List, and options for Content you can add to a track.

Add some content by clicking the add content button to open the menu. Then click any of the options to add content to the content slots.

Beat Visualizer

The beat visualizer uses volume amplitude and peak detection to detect beats and changes in the music.

The threshold for beat detection can be adjusted to make it more or less sensitive or to handle different volume levels. The threshold setting is located in the Track Settings panel, so you can have different thresholds per track.

Content can change on the beat, every other beat, the measure, or every other measure. This option is in the Track Settings panel.

Content Blocks

You can compose a set or track with different types of visual content. These content types are called content blocks. When you click on a type of conent in the left sidebar it will be added to the track's content slots. Settings for individual content blocks can be found in the Content Settings panel which is visible when you select a block in the track's content slots.

There are several types of content blocks in VISUALZ:

Image - Just like it sounds you can upload any image you want in your set/track.

Video - Upload any video to your set/track.

Webcam - Connect your webcam to include shots of the artist or the crowd.

Color - Very simple color content.

Text - Need some text on the screen? That's what this is for.

Bubbles - Bubbles effect. This is one of the first code generated effects in the system. There will be many more code based effects added as new versions release.

Oscilliscope - An actual oscilliscope code generated effect that responds to the actual audio frequencies in the music.

Gif - The gif block will grab a random gif and insert it into the set. You can type in a search tag to get gifs related to any subject you want.
