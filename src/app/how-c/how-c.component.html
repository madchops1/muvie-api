Set Toolbar

Cloud Upload

Uploads all local media files in set to the cloud and replaces the paths in the set file with the cloud paths. Helps sharing set files.

Open Browser Live Stream

Stream the contents of the preview window directly to the browser so you can perform live visuals remotely for events or OBS streams.

Open External Window

Open multiple external windows for multiple displays/windows and for easy use of the main UI during live performances.

Fullscreen

Go fullscreen in the main window. Use this for projecting when you have to mirror a single display. Change tracks using the remote control from your phone.

Set Preferences

Open the set preferences window.

Add Track

Add a track to your set.
